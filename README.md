# Convert txt to json

## Example txt file contents
```txt
hello
my
name
is
yongyong
```

## Convert
### Change prams
1. txtFile_name
2. jsonFile_path
3. file_format

```json
{
  "./jsonFile_path/1": "hello",
  "./jsonFile_path/2": "my",
  "./jsonFile_path/3": "name",
  "./jsonFile_path/4": "is",
  "./jsonFile_path/5": "yongyong"
}
```

### Run
- python3 txtTojson.py
